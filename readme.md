# Simple Cab Booking Service (Backend)

### Setup

**Requires MongoDB**

```shell
npm install
```

Run Server

```shell
node server.js
```


Server runs on a port Specified in the ```.env``` file navigate to ```localhost:PORT``` in the browser

## Routes (all POST)
```
├── addAPI/
│   ├── addUser
│   ├── addCab
├── bookAPI/
│   ├── book
│   ├── getCabs
│   ├── tripDone
│   ├── getTo
│   ├── getFrom


```

Check ```handlers/``` for detailed information
