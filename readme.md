# Cab Booking Service (Backend)

### Setup
```shell
npm install
```

Run Server

```shell
node server.js
```

Server runs on a port Specified in the ```.env``` file

## Routes (all POST)
```
├── addAPI/
│   ├── addUser
│   ├── addCab
├── bookAPI/
│   ├── book
│   ├── getCabs
│   ├── tripDone

```

Check ```handlers/``` for detailed information
